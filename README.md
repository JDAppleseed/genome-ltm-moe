# genome-ltm-moe

Research scaffolding for a Genome Long-Term Memory Mixture-of-Experts (LTM-MoE) concept. This repository captures planning docs, example schemas, configuration sketches, and lightweight scripts used during exploratory work.

## Contents
- `docs/`: Planning notes and system design documents.
- `configs/`: Model and data configuration stubs.
- `schemas/`: JSON schema definitions for internal data contracts.
- `scripts/`: Utility scripts and checks.
- `examples/`: Example inputs and artifacts.

## Status
This repository is a draft reference and is **not** production-ready.
